## Comment utilisé ce dépot:

- En ce qui concerne la BDD, pour déployé en local il sera important de créer les tables avec le fichier "Parrot.sql" en premier.

- Ensuite il faudra créér la table Roles et Utilisateurs avec account de MR parrot admin du site avec le fichiez (CréationCompteMrParrot.php)

**Il sera important utilisé le fichiez sur un lecteur PHP(wamp, mamp ou autre ) car cela intégrera en BDD le Hash du password.**

De plus il sera important avant de rentré les données local ou distant selon comment vous allez installé votre base de donnée.
